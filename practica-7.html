🟦 Práctica 7

Crear documentos JSON desde formularios

Generar documentos dinámicos desde una interfaz real.

📘 Guía de trabajo

El alumno construirá un formulario HTML que capture datos y los convierta en un documento JSON para enviarlo a Firestore, reforzando la idea de que la interfaz produce datos estructurados.

Un formulario bien diseñado reduce errores de captura y mejora la calidad del JSON que guardas.

🎯 Objetivos de aprendizaje

  • Traducir campos de formulario a campos JSON.
  • Validar entradas básicas antes de guardar.
  • Verificar en Firestore que el documento quedó correcto.

💻 Actividades en laboratorio

  • Crear un formulario HTML con al menos 5 campos.
  • Construir un objeto JSON con los datos capturados.
  • Enviar el JSON a Firestore.
  • Comprobar el resultado en la consola.

🧪 Practicario (entregable)

  • Formulario funcionando.
  • Al menos 3 envíos exitosos con datos diferentes.
  • Capturas del formulario y del documento en Firestore.

🧩 Recomendaciones

  • Usa nombres de campos claros y consistentes.
  • Evita guardar números como texto cuando no sea necesario.
  • Incluye un campo de fecha o marca de tiempo si aplica.
