🟦 Práctica 2

Tutorial: Catálogo Web de Películas con MongoDB Atlas + Flet (sample_mflix)

🧠 Objetivo de aprendizaje

  • Conectarse a MongoDB Atlas usando PyMongo + TLS (certifi).
  • Verificar conexión con ping.
  • Realizar consultas find_one() y find() con proyección, orden y límite.
  • Construir una app web en Flet con dropdown + panel de detalle.
  • Comprender el flujo UI → evento → detalle sin reconsulta a BD.

⏱️ Duración sugerida

100 minutos (2 bloques de 50 min o 1 sesión larga).

✅ Requisitos

  • Python 3.10 o superior.
  • Internet estable.
  • URI de MongoDB Atlas funcional.
pip install pymongo certifi flet

1) PRUEBA DE CONEXIÓN (SCRIPT DE CONSOLA)

1.1 Importaciones

from pymongo import MongoClient
import certifi

¿Qué hace cada una?

  • MongoClient: crea el cliente de conexión a MongoDB (Atlas o local).
  • certifi: trae certificados raíz (CA) confiables.
  • Atlas usa TLS; Python necesita confiar en una CA para validar el certificado del servidor.
  • certifi.where() da la ruta de cacert.pem.

1.2 Cadena de conexión (URI)

uri = "mongodb+srv://luisalfonsomartinez_db_user:LuisAtlas2026@cluster0.huc0clk.mongodb.net/sample_mflix?retryWrites=true&w=majority"

¿Qué significa?

  • mongodb+srv://: usa DNS SRV para descubrir nodos del clúster.
  • usuario:password@cluster...: credenciales de autenticación.
  • /sample_mflix: base por defecto al conectar.
  • retryWrites=true: reintenta escrituras en fallos temporales.
  • w=majority: confirma escritura por mayoría.

1.3 Crear el cliente de MongoDB

client = MongoClient(
    uri,
    serverSelectionTimeoutMS=5000,
    tls=True,
    tlsCAFile=certifi.where()
)

Explicación parámetro por parámetro

  • uri: cadena de conexión.
  • serverSelectionTimeoutMS=5000: máximo 5 segundos para encontrar servidor.
  • tls=True: fuerza conexión cifrada.
  • tlsCAFile=certifi.where(): archivo CA para validar certificado Atlas.

1.4 Bloque try/except

try:
    client.admin.command("ping")
    print("✅ Conectado a Atlas con certificado OK")

    db = client["sample_mflix"]
    col = db["movies"]
    doc = col.find_one({}, {"title": 1})
    print("🎬 Ejemplo:", doc.get("title"))

except Exception as e:
    print("❌ Error:")
    print(e)

1.4.1 client.admin.command("ping")

ping confirma conectividad real, TLS funcional y respuesta del servidor Atlas.

1.4.2 Seleccionar base y colección

db = client["sample_mflix"]
col = db["movies"]

MongoDB no exige crear colección antes para leer; si no existe, se crea al insertar.

1.4.3 Consulta find_one

doc = col.find_one({}, {"title": 1})
  • Filtro {}: sin filtro, trae cualquiera.
  • Proyección {"title": 1}: trae title y _id por defecto.

1.4.4 Mostrar resultado

print("🎬 Ejemplo:", doc.get("title"))

1.4.5 Manejo de error

Si falla DNS, contraseña, red, firewall o TLS, imprime el error real.

2.5 a 2.15 Código completo de la app

from datetime import datetime

import certifi
import flet as ft
from pymongo import MongoClient
from pymongo.errors import ConfigurationError


URI_POR_DEFECTO = "mongodb+srv://luisalfonsomartinez_db_user:LuisAtlas2026@cluster0.huc0clk.mongodb.net/sample_mflix?retryWrites=true&w=majority"
BASE_DATOS_POR_DEFECTO = "sample_mflix"
COLECCION_POR_DEFECTO = "movies"


def formatear_fecha(valor):
    if isinstance(valor, datetime):
        return valor.strftime("%Y-%m-%d")
    if isinstance(valor, str) and valor:
        return valor[:10]
    return "-"


def etiqueta(texto: str, color: str = ft.Colors.BLUE_50):
    return ft.Container(
        content=ft.Text(texto, size=11),
        padding=ft.Padding.symmetric(horizontal=10, vertical=4),
        border_radius=999,
        bgcolor=color,
    )


def principal(pagina: ft.Page):
    pagina.title = "Catalogo mflix"
    pagina.theme_mode = ft.ThemeMode.LIGHT
    pagina.padding = 16
    pagina.scroll = "auto"

    uri = URI_POR_DEFECTO
    nombre_bd = BASE_DATOS_POR_DEFECTO
    nombre_coleccion = COLECCION_POR_DEFECTO

    cliente = None
    coleccion = None

    def notificar(mensaje: str):
        pagina.snack_bar = ft.SnackBar(ft.Text(mensaje))
        pagina.snack_bar.open = True
        pagina.update()

    def obtener_coleccion():
        nonlocal cliente, coleccion
        if coleccion is not None:
            return coleccion

        if not uri:
            raise ValueError("Falta URI de conexion a MongoDB.")

        cliente = MongoClient(
            uri,
            tls=True,
            tlsCAFile=certifi.where(),
            serverSelectionTimeoutMS=7000,
        )
        cliente.admin.command("ping")
        coleccion = cliente[nombre_bd][nombre_coleccion]
        return coleccion

    documentos_por_id = {}
    desplegable_pelicula = ft.Dropdown(label="Selecciona una pelicula", width=500)
    contenedor_detalle = ft.Container()
    texto_info = ft.Text(size=12, opacity=0.75)

    def construir_detalle(documento):
        titulo = documento.get("title") or "(sin titulo)"
        id_pelicula = str(documento.get("_id", "-"))
        anio = documento.get("year") or "-"
        clasificacion = documento.get("rated") or "-"
        duracion = documento.get("runtime") or "-"
        generos = documento.get("genres") or []
        idiomas = documento.get("languages") or []
        paises = documento.get("countries") or []
        reparto = documento.get("cast") or []
        directores = documento.get("directors") or []
        fecha_estreno = formatear_fecha(documento.get("released"))
        trama = documento.get("fullplot") or documento.get("plot") or "Sin descripcion"
        premios = documento.get("awards") or {}
        imdb = documento.get("imdb") or {}
        tomatoes = documento.get("tomatoes") or {}
        poster = documento.get("poster")

        calificacion_imdb = imdb.get("rating", "-")
        votos_imdb = imdb.get("votes", "-")
        calificacion_tomatoes = "-"
        if isinstance(tomatoes, dict):
            viewer = tomatoes.get("viewer", {})
            calificacion_tomatoes = viewer.get("rating", "-") if isinstance(viewer, dict) else "-"

        texto_premios = premios.get("text", "-") if isinstance(premios, dict) else "-"

        imagen = (
            ft.Image(src=poster, width=260, height=380)
            if isinstance(poster, str) and poster.startswith("http")
            else ft.Container(
                content=ft.Icon(ft.Icons.IMAGE_NOT_SUPPORTED, size=60, opacity=0.5),
                width=260,
                height=380,
                alignment=ft.Alignment(0, 0),
                bgcolor=ft.Colors.GREY_100,
                border_radius=10,
            )
        )

        return ft.Container(
            content=ft.Card(
                content=ft.Container(
                    padding=12,
                    content=ft.Row(
                        spacing=16,
                        vertical_alignment=ft.CrossAxisAlignment.START,
                        controls=[
                            ft.Container(content=imagen, border_radius=10),
                            ft.Container(
                                expand=True,
                                content=ft.Column(
                                    spacing=8,
                                    tight=True,
                                    controls=[
                                        ft.Text(f"{titulo} ({anio})", size=22, weight=ft.FontWeight.W_600),
                                        ft.Row(
                                            wrap=True,
                                            controls=[
                                                etiqueta(f"ID: {id_pelicula}"),
                                                etiqueta(f"Clasificacion: {clasificacion}"),
                                                etiqueta(f"Duracion: {duracion} min"),
                                                etiqueta(f"Estreno: {fecha_estreno}"),
                                            ],
                                        ),
                                        ft.Row(
                                            wrap=True,
                                            controls=[etiqueta(g) for g in generos]
                                            if generos
                                            else [etiqueta("Sin genero", ft.Colors.GREY_200)],
                                        ),
                                        ft.Text(trama, selectable=True),
                                        ft.Divider(height=4),
                                        ft.Text(
                                            f"Directores: {', '.join(directores) if directores else '-'}",
                                            size=12,
                                        ),
                                        ft.Text(
                                            f"Reparto: {', '.join(reparto[:12]) if reparto else '-'}",
                                            size=12,
                                        ),
                                        ft.Text(
                                            f"Idiomas: {', '.join(idiomas) if idiomas else '-'}",
                                            size=12,
                                        ),
                                        ft.Text(
                                            f"Paises: {', '.join(paises) if paises else '-'}",
                                            size=12,
                                        ),
                                        ft.Text(f"Premios: {texto_premios}", size=12),
                                        ft.Text(
                                            f"IMDb: {calificacion_imdb} ({votos_imdb} votos)",
                                            size=12,
                                        ),
                                        ft.Text(
                                            f"Tomatoes espectador: {calificacion_tomatoes}",
                                            size=12,
                                        ),
                                        ft.Text(
                                            f"Comentarios mflix: {documento.get('num_mflix_comments', 0)}",
                                            size=12,
                                        ),
                                    ],
                                ),
                            ),
                        ],
                    ),
                ),
            ),
        )

    def al_seleccionar_pelicula(evento):
        id_pelicula = str(desplegable_pelicula.value).strip() if desplegable_pelicula.value else ""
        if not id_pelicula:
            contenedor_detalle.content = ft.Text("Selecciona una pelicula para ver su detalle.")
        else:
            documento_seleccionado = documentos_por_id.get(id_pelicula)
            if documento_seleccionado is None:
                contenedor_detalle.content = ft.Text(
                    "No se encontro la pelicula seleccionada. Recarga la lista."
                )
            else:
                contenedor_detalle.content = construir_detalle(documento_seleccionado)
        pagina.update()

    desplegable_pelicula.on_select = al_seleccionar_pelicula

    def cargar_peliculas():
        try:
            coleccion_db = obtener_coleccion()
        except Exception as error:
            mensaje_error = str(error)
            if isinstance(error, ConfigurationError) and "_mongodb._tcp" in mensaje_error:
                texto_info.value = (
                    "Error DNS con mongodb+srv. Revisa internet/DNS o usa URI estandar mongodb://"
                )
            else:
                texto_info.value = "Sin conexion a MongoDB"
            contenedor_detalle.content = ft.Text("No se pudo conectar con MongoDB.")
            pagina.update()
            notificar(str(error))
            return

        proyeccion = {
            "title": 1,
            "year": 1,
            "poster": 1,
            "genres": 1,
            "plot": 1,
            "fullplot": 1,
            "languages": 1,
            "released": 1,
            "directors": 1,
            "rated": 1,
            "awards": 1,
            "runtime": 1,
            "cast": 1,
            "countries": 1,
            "imdb": 1,
            "tomatoes": 1,
            "num_mflix_comments": 1,
        }
        documentos = list(coleccion_db.find({}, proyeccion).sort("title", 1).limit(300))
        documentos_por_id.clear()
        desplegable_pelicula.options = []

        for documento in documentos:
            id_pelicula = str(documento.get("_id"))
            titulo = documento.get("title") or "(sin titulo)"
            anio = documento.get("year") or "-"
            documentos_por_id[id_pelicula] = documento
            desplegable_pelicula.options.append(
                ft.dropdown.Option(key=id_pelicula, text=f"{titulo} ({anio})")
            )

        texto_info.value = f"Peliculas cargadas: {len(documentos)}"
        desplegable_pelicula.value = None
        if documentos:
            contenedor_detalle.content = ft.Text("Selecciona una pelicula para ver su detalle.")
        else:
            contenedor_detalle.content = ft.Text("No hay peliculas para mostrar.")

        desplegable_pelicula.update()
        pagina.update()

    pagina.appbar = ft.AppBar(
        title=ft.Text("Catalogo de peliculas mflix"),
        actions=[
            ft.IconButton(
                icon=ft.Icons.REFRESH,
                tooltip="Recargar",
                on_click=lambda e: cargar_peliculas(),
            )
        ],
    )

    pagina.add(
        desplegable_pelicula,
        texto_info,
        contenedor_detalle,
    )

    cargar_peliculas()


ft.run(principal, view=ft.AppView.WEB_BROWSER)

4) Problemas comunes y soluciones (modo escuela)

  • Error: _mongodb._tcp → problema DNS/SRV. Cambiar red, probar hotspot o DNS público.
  • Error: Authentication failed → revisar usuario, contraseña y roles en Atlas.
  • Error: IP not allowed → agregar IP actual en Network Access de Atlas.
