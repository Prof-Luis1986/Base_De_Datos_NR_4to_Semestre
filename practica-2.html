🟦 Práctica 2

Captura de datos y creación de documentos JSON usando comandos en MongoDB

🧠 Guía de trabajo

En esta práctica aprenderás a capturar información y transformarla en documentos JSON, utilizando comandos de MongoDB ejecutados desde MongoDB Studio 3T.

El objetivo es que comprendas cómo se crea una base de datos no relacional desde cero, cómo se insertan documentos JSON mediante comandos y cómo estos documentos forman parte de una colección. Antes de usar herramientas visuales o servicios en la nube, es indispensable entender qué hace cada comando y qué efecto tiene sobre los datos.

🎯 Aprendizajes esperados

Al finalizar la práctica, serás capaz de:

  • Crear una base de datos y una colección mediante comandos.
  • Insertar múltiples documentos JSON usando insertMany().
  • Consultar documentos con find().
  • Comprender cómo MongoDB almacena documentos JSON.
  • Relacionar comandos con cambios reales en la base de datos.

🪑 Actividades en salón

(Captura y análisis – sin computadora)

1. Captura de información

Se plantea el siguiente contexto: registro básico de usuarios de un sistema.

En tu cuaderno, captura la información de al menos 5 personas:

  • Nombre
  • Apellido
  • Edad

No pienses aún en MongoDB ni en comandos. Solo recopila los datos de forma clara.

2. Conversión de datos a estructura JSON

Con los datos capturados, responde:

  • ¿Qué datos se repiten?
  • ¿Qué campos serán las claves?
  • ¿Qué tipo de dato tiene cada valor?

Con base en esto, escribe un JSON base en tu cuaderno.

Ejemplo de referencia:

{
  "nombre": "",
  "apellido": "",
  "edad": 0
}
📌 Producto del salón: Datos capturados + JSON base definido.

💻 Actividades en laboratorio

(MongoDB Studio 3T – comandos)

Parte A – Creación de base y colección

Abre Studio 3T y crea una nueva ventana de consulta. Ejecuta los siguientes comandos:

use registro_nombres;

// Crea la colección
 db.createCollection("usuarios");

Verifica visualmente que la base y la colección existan.

Parte B – Inserción de documentos JSON

Inserta los documentos usando insertMany():

db.usuarios.insertMany([
  { nombre: "Juan", apellido: "Pérez", edad: 30 },
  { nombre: "María", apellido: "García", edad: 25 },
  { nombre: "Carlos", apellido: "López", edad: 40 }
]);

Consulta los documentos:

db.usuarios.find().pretty();

🧪 Practicario

Parte A – Comprensión de comandos

Explica con tus palabras qué hace cada comando:

  • use
  • createCollection
  • insertMany
  • find().pretty()

Parte B – Inserción ampliada

  • Inserta al menos 5 documentos adicionales, usando los datos capturados en el salón.
  • Ejecuta nuevamente: db.usuarios.find().pretty();

Responde:

  • ¿Cuántos documentos hay ahora?
  • ¿Todos tienen la misma estructura?

Parte C – Análisis de estructura

Identifica:

  • ¿Qué campos se repiten?
  • ¿Qué campo identifica de forma única a cada documento?

Explica por qué MongoDB genera automáticamente el campo _id.
