🟦 Práctica 3

Consulta y eliminación de documentos JSON usando comandos en MongoDB

🧠 Guía de trabajo

En esta práctica aprenderás a consultar y eliminar documentos JSON usando comandos de MongoDB. Comprenderás cómo los filtros permiten localizar información específica y por qué eliminar datos requiere criterio y precisión, especialmente cuando se trabaja con identificadores únicos como _id.

🎯 Aprendizajes esperados

Al finalizar la práctica, serás capaz de:

  • Consultar documentos completos y filtrados.
  • Eliminar documentos usando condiciones.
  • Eliminar documentos usando _id.
  • Analizar el impacto de borrar información.
  • Explicar por qué _id es el identificador más seguro.

🪑 Actividades en salón

(Análisis conceptual)

  • ¿Qué pasaría si borraras un registro incorrecto?
  • ¿Es seguro borrar usando solo un nombre?
  • Analiza este filtro: { "nombre": "Juan" }

💻 Actividades en laboratorio

(MongoDB Studio 3T – comandos)

Parte A – Consultas

db.usuarios.find().pretty();

db.usuarios.find({ nombre: "Juan" }).pretty();

Parte B – Eliminación por campo

db.usuarios.deleteOne({ nombre: "Juan" });

db.usuarios.find().pretty();

Intenta borrar de nuevo a “Juan” y observa el resultado.

Parte C – Eliminación por _id

db.usuarios.find().pretty();

// Copia un _id real
 db.usuarios.deleteOne({ _id: ObjectId("PEGA_AQUI_EL_ID_REAL") });

 db.usuarios.find().pretty();

🧪 Practicario

Parte A – Análisis crítico

Responde:

Parte B – Evidencia de comandos

  • Después de borrar
