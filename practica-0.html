🟨 Práctica 0

Fundamentos de Bases de Datos No Relacionales

Modalidad: Exposición + Podcast · Duración: 8 horas (fuera de laboratorio)

🎯 Propósito

Comprender el origen, la importancia y las características de las bases de datos No Relacionales (NoSQL), desarrollando la capacidad de explicar conceptos técnicos de manera clara y fundamentada.

📚 Actividad 1 – Investigación Guiada

Investigar y organizar información sobre:

  • ¿Qué es una base de datos?
  • ¿Qué es una base de datos relacional (SQL)?
  • ¿Qué es una base de datos No Relacional (NoSQL)?
  • Diferencias principales entre SQL y NoSQL.
  • ¿Qué significa escalabilidad horizontal?
  • Ejemplos reales de uso (Netflix, Amazon, redes sociales, videojuegos).

🎙 Actividad 2 – Podcast Académico

Tema: “¿Por qué NoSQL cambió la forma de almacenar datos?”

Requisitos:

  • Trabajo en equipos.
  • Duración: 5 a 8 minutos.
  • Grabado con celular.
  • Audio claro y organizado.
  • Explicación natural (no lectura completa).

Estructura obligatoria del podcast:

  • Presentación del equipo.
  • Explicación clara de SQL.
  • Explicación clara de NoSQL.
  • Comparación técnica.
  • Ejemplo real.
  • Conclusión argumentada del equipo.

🎤 Actividad 3 – Exposición Presencial

Cada equipo realizará una exposición de 8–10 minutos donde deberá presentar:

  • Un mapa conceptual o diapositiva.
  • Un ejemplo de documento JSON explicado.
  • Un ejemplo ilustrado de un tipo de base NoSQL (grafo, documental, etc.).
  • Participación de todos los integrantes.

📦 Entregables

Importante: Todo debe adjuntarse en su Google Sites.

  • Archivo de audio en formato mp3 o similar.
  • Evidencia de exposición.
  • Material visual utilizado.
